# Microsoft Entra ID (Azure AD) IAM Labs

Hands-on Microsoft Entra ID labs demonstrating identity lifecycle management, access control, RBAC, MFA, and authentication security configurations.

---

## Completed Labs

### 1️⃣ User Management
- Create users
- Reset passwords
- Manage account properties
- Verify user lifecycle actions

### 2️⃣ Security Group Management
- Create security groups
- Add members
- Assign directory roles to groups
- Verify role assignments

### 3️⃣ Multi-Factor Authentication (MFA)
- Enable per-user MFA
- Enforce MFA
- Reset password before enforcement
- Verify MFA status

---

## Skills Demonstrated

- Identity and Access Management (IAM)
- Microsoft Entra ID Administration
- Role-Based Access Control (RBAC)
- Multi-Factor Authentication (MFA)
- Identity Security Hardening
